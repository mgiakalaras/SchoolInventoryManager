# School Inventory Manager

Αρχική έκδοση MVP για σχολική απογραφή Η/Υ και ψηφιακών μέσων.

## Τι έχει ήδη

- ASP.NET Core Razor Pages (.NET 8)
- SQLite βάση δεδομένων
- Στοιχεία σχολείου / σχολικό έτος / υπεύθυνος απογραφής
- Χώροι σχολείου
- Εξοπλισμός ανά χώρο
- Κατηγορίες εξοπλισμού
- Κατάσταση εξοπλισμού: λειτουργικό, χρειάζεται έλεγχο, χαλασμένο, προς απόσυρση κλπ.
- Dashboard με βασικά σύνολα
- Αναφορά απογραφής σε μορφή HTML με εκτύπωση / Save as PDF
- Λογότυπο σχολείου

## Πώς το ανοίγεις στο Visual Studio

1. Κάνε extract το ZIP.
2. Άνοιξε το αρχείο `SchoolInventoryManager.csproj` με Visual Studio 2022 ή νεότερο.
3. Περίμενε να γίνει Restore στα NuGet packages.
4. Πάτα F5 ή Ctrl+F5.
5. Η βάση `school_inventory.db` δημιουργείται αυτόματα στην πρώτη εκκίνηση.

## Χρήση

1. Άνοιξε `Σχολείο` και συμπλήρωσε στοιχεία σχολικής μονάδας.
2. Άνοιξε `Χώροι` και πρόσθεσε/διόρθωσε αίθουσες και γραφεία.
3. Άνοιξε `Εξοπλισμός` και καταχώρισε αντικείμενα.
4. Άνοιξε `Αναφορές` και πάτα `Προεπισκόπηση / Εκτύπωση PDF`.
5. Από τον browser κάνε `Ctrl + P` και επίλεξε `Save as PDF`.

## Επόμενα βήματα ανάπτυξης

- Άμεσο server-side PDF export με QuestPDF.
- Excel import/export με ClosedXML.
- Αναλυτικό template εισαγωγής Excel.
- QR codes ανά αντικείμενο.
- Φωτογραφίες αντικειμένων.
- Backup/restore.
- Χρήστες και ρόλοι.
- Ιστορικό αλλαγών.

## v5 - Import / Export

Η έκδοση v5 προσθέτει σελίδα **Import / Export**.

### Excel import

Η εφαρμογή διαβάζει αρχείο `.xlsx` με τις εξής στήλες:

- Χώρος
- Κατηγορία
- Αντικείμενο
- Ποσότητα
- Μάρκα
- Μοντέλο
- Serial Number
- Κατάσταση
- Περιγραφή
- Παρατηρήσεις

Υποχρεωτικές είναι μόνο οι στήλες **Χώρος** και **Αντικείμενο**. Αν η ποσότητα μείνει κενή, μπαίνει αυτόματα 1. Αν ο χώρος ή η κατηγορία δεν υπάρχουν, δημιουργούνται αυτόματα.

### Excel / CSV export

Από τη σελίδα **Import / Export** μπορείς να εξάγεις όλα τα ενεργά αντικείμενα σε Excel ή CSV, ώστε τα δεδομένα να μπορούν να χρησιμοποιηθούν και σε άλλες εφαρμογές.

### Dark theme

Το σκοτεινό θέμα είναι πλέον το προεπιλεγμένο θέμα της εφαρμογής. Η αναφορά εκτύπωσης παραμένει λευκή/καθαρή για σωστή εκτύπωση σε PDF ή χαρτί.
