# About / Contact refresh

## Purpose

Οι σελίδες About και Επικοινωνία ανανεώθηκαν ώστε να δείχνουν πιο επαγγελματικές και πιο καθαρές για χρήση σε δημόσιο repository ή σχολική δοκιμή.

## About

Περιλαμβάνει:

- σκοπό εφαρμογής
- κύριες δυνατότητες
- τεχνολογίες
- κατάσταση έργου
- δημιουργό
- έκδοση εφαρμογής
- links σε email, GitHub και repository

## Contact

Περιλαμβάνει:

- βασικό email επικοινωνίας
- links για υποστήριξη, bug report και feature request
- checklist για αναφορά προβλημάτων
- πρότυπο μηνύματος
- σημείωση ασφάλειας / school testing
