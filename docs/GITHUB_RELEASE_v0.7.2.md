# School Inventory Manager v0.7.2 - Adaptive layout, Scanner release link and About polish

This is a UI/UX server-test release for the central web application.

## What's new

- Added a global adaptive wide-layout CSS layer.
- Reduced wasted horizontal space on wide displays.
- Reworked the Scanner page.
- Scanner page now links directly to the Android Scanner GitHub Releases.
- Scanner page explains APK vs AAB.
- Reworked the About page as a more complete project presentation.
- Updated app version to `0.7.2`.

## Changed areas

- Global layout
- `/Scanner`
- `/About`
- App version/release docs

## No migration required

No database migration is required.

## Recommended before server update

- Backup the database/volume.
- Pull/redeploy the latest app image/stack.
- Test dashboard, Scanner page and About page.
- Hard refresh browser if old CSS appears.
