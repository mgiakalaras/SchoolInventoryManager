# School Inventory Manager

Web εφαρμογή για την απογραφή, τεχνική παρακολούθηση και διαχείριση ψηφιακού εξοπλισμού σε σχολικές μονάδες.

Η εφαρμογή βοηθά τον υπεύθυνο απογραφής / Πληροφορικής να καταχωρίζει χώρους, εξοπλισμό, τεχνικά χαρακτηριστικά, ανταλλακτικά, αναλώσιμα, αναφορές, καταστροφές υλικού και εκτυπώσιμα έντυπα από υπολογιστή, tablet ή κινητό μέσα στο τοπικό σχολικό δίκτυο.

> Τρέχουσα φάση: **School testing**  
> Προτεινόμενη χρήση: **τοπικό σχολικό δίκτυο**  
> Μη προτεινόμενη χρήση: **δημόσια έκθεση στο Internet χωρίς login / HTTPS / roles**

---

## Τι κάνει η εφαρμογή

Το School Inventory Manager ξεκίνησε ως απλή εφαρμογή απογραφής σχολικού εξοπλισμού και εξελίσσεται σε εργαλείο τεχνικής διαχείρισης σχολικής υποδομής.

Υποστηρίζει:

- χώρους σχολείου
- κατηγορίες εξοπλισμού
- καταχώριση υλικών ανά χώρο
- κατάσταση εξοπλισμού
- Excel import / export
- αναφορές απογραφής
- εκτύπωση / Save as PDF
- διαδικασία καταστροφής υλικού
- τεχνικά χαρακτηριστικά Η/Υ / laptop / mini PC / διαδραστικών
- βάση τεχνικών αναφορών για CPU, RAM, δίσκους, λειτουργικά κ.λπ.
- απόθεμα ανταλλακτικών και αναλώσιμων
- ιστορικό χρήσης ανταλλακτικών
- τεχνικό έλεγχο και υποψήφιες αναβαθμίσεις
- dashboard τύπου control center

## Import / Export

### Excel import

- Εισαγωγή από `.xlsx`
- Προεπισκόπηση εισαγωγής
- Αντιστοίχιση άγνωστων χώρων / κατηγοριών
- Υποστήριξη μαζικών εγγραφών
- Δυνατότητα διαχωρισμού εγγραφών που χρειάζονται ξεχωριστή παρακολούθηση
- Έλεγχος πιθανών διπλοεγγραφών
- Conflict review πριν την τελική εισαγωγή
- Επιλογές:
  - Παράλειψη
  - Ενημέρωση υπάρχουσας εγγραφής
  - Εισαγωγή ως νέα εγγραφή

### Duplicate guard

Η εφαρμογή προσπαθεί να αποφύγει διπλοεγγραφές σε επανεισαγωγή Excel.

Ελέγχει:

- serial number
- χώρο
- κατηγορία
- ονομασία αντικειμένου
- μάρκα
- μοντέλο
- λοιπά στοιχεία όπου υπάρχουν

Αν μια εγγραφή μοιάζει με ήδη υπάρχουσα, ο χρήστης μπορεί να αποφασίσει αν θα την παραλείψει, θα ενημερώσει την παλιά εγγραφή ή θα την περάσει ως νέα.

### Export

- Excel export
- CSV export
- Αναφορές απογραφής
- Τεχνικές αναφορές
- Απόθεμα ανταλλακτικών
- Εκτύπωση / Save as PDF από browser

## Docker / Portainer deployment

Η εφαρμογή μπορεί να τρέξει σε σχολικό server, ZimaOS, CasaOS, Linux server ή Windows μηχάνημα με Docker.

### Docker Compose

```bash
docker compose up -d --build
```

Άνοιγμα από άλλη συσκευή στο ίδιο δίκτυο:

```text
http://SERVER-IP:5148
```

Παράδειγμα:

```text
http://192.168.1.80:5148
```

### Portainer

Στο Portainer:

```text
Stacks → Add stack → Repository
```

Ρυθμίσεις:

```text
Name:
school-inventory-manager

Repository URL:
https://github.com/mgiakalaras/SchoolInventoryManager.git

Repository reference:
refs/heads/main

Compose path:
docker-compose.yml
```

Η εφαρμογή χρησιμοποιεί Docker named volume για τα δεδομένα:

```text
school_inventory_app_data
```
